# PPT 模板目录

将北信科官方 PPT 模板放入此目录。

## 推荐模板

- `BISTU通用PPT模板.pptx` — 北信科官方蓝色主题模板
- `BISTU答辩PPT模板.pptx` — 毕业答辩专用模板
- `BISTU汇报PPT模板.pptx` — 工作汇报模板

## 模板制作指南

1. 使用北信科官方配色：
   - 主题蓝: `#00468C` (RGB: 0, 70, 140)
   - 辅助灰: `#808080`
   - 背景白: `#FFFFFF`

2. 包含以下版式：
   - 封面页
   - 目录页
   - 内容页（标题+正文）
   - 图片页
   - 结束页

3. 每个版式命名清晰，便于 python-pptx 识别

## 获取模板

- 联系学校宣传部或教务处获取官方模板
- 或自行制作符合学校视觉规范的模板
